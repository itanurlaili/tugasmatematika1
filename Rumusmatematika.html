RUMUS MATEMATIKA

Sifat Komutatif Penjumlahan

Rumus : a + b = b + a

Contoh:2 + 4 = 4 + 2 = 6 atau 7 + 10 = 10 + 7 = 17

Sifat Komutatif Perkalian

Rumus : a x b = b x a

Contoh : 3 x 5 = 5 x 3 = 15 atau 20 x 2 = 2 x 20 = 40

Sifat Asosiatif Penjumlahan

Rumus : (a + b) + c = a + (b + c)

Contoh : (3 + 5) + 7 = 3 + (5 + 7) = 15 atau (4 + 3 ) + 10 = 4 + (3 + 10) = 17

Sifat Asosiatif Perkalian

Rumus : (a x b) x c = a x (b x c)

Contoh : (3 x 5) x 2 = 3 x (5 x 2) = 30 atau (12 x 2) x 10 = 12 x (2 x 10) = 240

Sifat Distributif Perkalian pada Penjumlahan

Rumus : a x (b + c) = (a x b) + (a x c)

Contoh : 2 x (5 + 10) = (2 x 5) + (2 x 10) = 10 + 20 = 30

Sifat Distributif Perkalian pada Pengurangan

Rumus : a x (b – c) = (a x b) – (a x c)

Contoh : 2 x (10 – 5) = (2 x 10) – (2 x 5) = 20 – 10 = 10
